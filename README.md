<h1> A AI Native Code Reviewer Tool </h1>
<p>A AI native code reviwer with vertical Integration of codebase upon each merge using Github webhooks,Qdrant vector database for code chunking strategy. With smart chunking especially for golang based project and fixed size chunking for any codebase other than golang</p>

**Features**

1. Integration with github
2. Smart chunking of codebases
3. quick installation using github
4. GROQ LLM integration for code reviewer and suggestions
5. Remembers Architectural Discussions.
6. Have Knowlege of the Discussions happened during each issue and pull request comment thread
7. Answer questions using mentions in the issues or Pull request thread
